# base case  &&  传递结束条件
- 指的是递归中不可再往前递推的节点，如斐波那契的f(1),f(2)。也就是开始回归的节点
- 当递归函数未执行完时会被暂时存放在系统栈（后进先出）一旦后续节点传回了所有值就会被调出来继续执行，执行完毕后被清除
- 话递归调用图去分析递归
# master公式 估计递归复杂度
- 所有子问题规模相同才能用
- a:递归分支数
- b:每个分支内的数据缩小的倍数
- c:除递归调用外函数中数据处理的复杂度$$T(N)=a*T(n/b)+o(n^c)$$
	1. log(b,a) < c, o(n^c);
	2. log(b,a) > c, o(n^log(b,a));
	3. log(b,a) == c, o(n^c\*logn);
	4. 特别的，T(n)=2\*T(n/2)+o(n\*logn), o(n\*(logn)^2)
