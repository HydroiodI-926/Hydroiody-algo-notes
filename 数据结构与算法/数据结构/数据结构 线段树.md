# 线段树（Segment Tree）

## 核心思想

线段树是一种**区间二分 + 归并分治**的树形结构，用于高效维护区间信息。每次操作将区间对半拆分，递归处理子区间后合并结果，单次操作 O(log n)。

## 存储结构

- **逻辑结构**：完全二叉树，根节点管理 $[1,n]$，每个节点 i 的左子为 $2i$、右子为 $2i+1$
- **物理结构**：用数组实现堆式存储，区间与下标一一对应，由递归参数 $(l,r,i)$ 维护，不需要额外记录
- **空间分析**：理论最大节点编号为 $4n$（树高 $h=\lceil \log_2 n \rceil + 1$，满二叉树节点数 $2^h-1 \leq 4n-1$）。由于数组下标从 0 开始，需要能访问到索引 $4n$，因此实际开 `n<<2|1`（即 $4n+1$），稳妥起见可开 $4n+5$

```cpp
// 三个核心数组：区间和、原始数据、懒标记缓存
vector<ll> sum, arr, lazy;
```

## 三个核心操作

线段树的所有修改围绕三种方向的操作展开：

| 操作 | 方向 | 时机 | 作用 |
|------|------|------|------|
| **up**（上传） | 底 → 顶 | 左右子区间处理完毕后 | 用子节点信息合并出父节点信息 |
| **apply**（直接修改） | 本层 | 当前区间被操作区间完全包含时 | 更新当前节点 + 挂载懒标记 |
| **down**（下传） | 顶 → 底 | 需要深入子区间之前 | 将当前节点的懒标记下推到左右儿子 |

```cpp
// 上传：用左右儿子的和更新当前节点
void up(int i) {
    sum[i] = sum[i << 1] + sum[i << 1 | 1];
}

// 直接修改：更新区间和 + 挂载懒标记（不继续向下）
void apply(int i, ll v, int n) {
    sum[i] += v * n;   // 每个元素 +v，区间长度 n
    lazy[i] += v;      // 记录缓存，后续需要时再下传
}

// 下传：将当前节点的懒标记推给左右儿子
void down(int i, int ln, int rn) {
    if (lazy[i] != 0) {
        apply(i << 1,     lazy[i], ln);
        apply(i << 1 | 1, lazy[i], rn);
        lazy[i] = 0;      // 清除当前层缓存
    }
}
```

## 懒标记机制（Lazy Propagation）

**核心原则**：只更新必要的层，其余层挂缓存，用到时再下传。

- 修改/查询某个区间时，如果当前区间被**完全包含**，直接在本层 `apply`，子区间暂不更新
- 当需要**深入子区间**（修改/查询的子区间与当前区间部分重叠）时，先 `down` 清掉当前层的缓存，然后再递归
- 下传只推**一层**，不会递归到底——下次用到下一层时，那一层会继续下推

> 一句话总结："能偷懒就偷懒，被查到了再还债"。

## 建树

```cpp
void build(int l, int r, int i) {
    if (l == r) {
        sum[i] = arr[l];        // 叶子节点直接赋初值
    } else {
        int mid = (l + r) >> 1;
        build(l, mid, i << 1);
        build(mid + 1, r, i << 1 | 1);
        up(i);                  // 回溯时合并
    }
    lazy[i] = 0;                // 初始化懒标记
}
```

## 区间修改

```cpp
void add(int jl, int jr, ll v, int l, int r, int i) {
    if (jl <= l && r <= jr) {
        // 当前区间被修改区间完全包含 → 直接改、挂缓存
        apply(i, v, r - l + 1);
    } else {
        int mid = (l + r) >> 1;
        down(i, mid - l + 1, r - mid);  // 需要深入 → 先清缓存
        if (jl <= mid) add(jl, jr, v, l, mid, i << 1);
        if (jr > mid)  add(jl, jr, v, mid + 1, r, i << 1 | 1);
        up(i);                          // 回溯时合并
    }
}
```

## 区间查询

> ⚠️ `query` 中也需要 `down`！如果查询前有未下传的懒标记，直接用 `sum[i]` 会读到过期的和。

```cpp
ll query(int jl, int jr, int l, int r, int i) {
    if (jl <= l && r <= jr) {
        return sum[i];                  // 完全包含 → 直接返回
    }
    int mid = (l + r) >> 1;
    down(i, mid - l + 1, r - mid);      // 需要深入 → 先清缓存
    ll ans = 0;
    if (jl <= mid) ans += query(jl, jr, l, mid, i << 1);
    if (jr > mid)  ans += query(jl, jr, mid + 1, r, i << 1 | 1);
    return ans;
}
```

## 完整模板

```cpp
#include <bits/stdc++.h>
typedef long long ll;
using namespace std;

vector<ll> arr, sum, lazy;

void up(int i) {
    sum[i] = sum[i << 1] + sum[i << 1 | 1];
}

void apply(int i, ll v, int n) {
    sum[i] += v * n;
    lazy[i] += v;
}

void down(int i, int ln, int rn) {
    if (lazy[i] != 0) {
        apply(i << 1,     lazy[i], ln);
        apply(i << 1 | 1, lazy[i], rn);
        lazy[i] = 0;
    }
}

void build(int l, int r, int i) {
    if (l == r) { sum[i] = arr[l]; }
    else {
        int mid = (l + r) >> 1;
        build(l, mid, i << 1);
        build(mid + 1, r, i << 1 | 1);
        up(i);
    }
    lazy[i] = 0;
}

void add(int jl, int jr, ll v, int l, int r, int i) {
    if (jl <= l && r <= jr) {
        apply(i, v, r - l + 1);
    } else {
        int mid = (l + r) >> 1;
        down(i, mid - l + 1, r - mid);
        if (jl <= mid) add(jl, jr, v, l, mid, i << 1);
        if (jr > mid)  add(jl, jr, v, mid + 1, r, i << 1 | 1);
        up(i);
    }
}

ll query(int jl, int jr, int l, int r, int i) {
    if (jl <= l && r <= jr) return sum[i];
    int mid = (l + r) >> 1;
    down(i, mid - l + 1, r - mid);
    ll ans = 0;
    if (jl <= mid) ans += query(jl, jr, l, mid, i << 1);
    if (jr > mid)  ans += query(jl, jr, mid + 1, r, i << 1 | 1);
    return ans;
}

// 用法：sum.resize(n << 2 | 1); lazy.resize(n << 2 | 1); build(1, n, 1);
```

## 变体速查

| 需求      | 改动点                                                 |
| ------- | --------------------------------------------------- |
| 区间最大值   | `up` 改为 `max(...)`，`apply` 改为 `sum[i] += v`（最大值直接加） |
| 区间最小值   | `up` 改为 `min(...)`，初始化为 INF                         |
| 同时维护多信息 | `sum` 改为结构体数组，`up` 中逐字段合并                           |
| 不修改、只查询 | 去掉 `lazy` / `apply` / `down`，更简单                    |
