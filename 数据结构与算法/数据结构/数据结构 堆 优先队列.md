# 堆 二叉堆
- 优先队列最稳定的实现就是通过堆，二叉堆从定义上是一颗**完全二叉树**
- 根据堆序性 二叉堆分为**大根堆**和**小根堆**
	- 大根堆：是指在整棵树中，**任意一个节点的值都大于或等于其子节点的值**
	- 小根堆：是指在整棵树中，**任意一个节点的值都小于或等于其子节点的值**
- 根据大根堆/小根堆的特点，实现[[排序 堆排序]]
## 用数组实现完全二叉树的排列方法构造堆
- 根据顺序从上到下，从左到右紧密构建，数组下标与该元素对应的在完全二叉树的位置规则如下：
- 第i节点的父节点为（i-1)/2；0节点的父节点是自己
- 第i节点的左子节点 i\*2+1,右子节点i\*2+2(如果>=size则不存在)
- 每次往堆中插入元素时，为维持堆序性，都必会经历向上调整或向下调整之一
```cpp
vector<int> heap;

// 向上调整
void heapinsert(int i){
    while(heap[i]>heap[(i-1)/2]){
        swap(heap[i],heap[(i-1)/2]);
        i=(i-1)/2;
    }
}

// 向下调整
void heapify(int i,int len){
    int l=i*2+1;
    // 确定左子节点存不存在
    while(l<len){ 
	    // 判断右子节点存不存在，并比较哪个孩子最大

        best=heap[best]>heap[i]?best:i;
        // 当父节点大于等于子节点

            break;
        }
        swap(heap[i],heap[best]);
        i=best;
        l=i*2+1;
    }
}
```
