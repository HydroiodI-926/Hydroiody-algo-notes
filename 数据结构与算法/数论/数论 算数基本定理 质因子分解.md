# 算数基本定理（唯一分解定理）
**任何大于1的正整数n都可以唯一分解为有限个素数的乘积**
$$n = p1^c1 * p2^c2 * ... * pm^cm$$
p1<p2<....\<pm是质数；c1,c2,c3....cm是正整数
# 质因数分解板子
```cpp
void f(int n){
	for(int j=2;j*j<=n;j++){
		if(n%j==0){
			printf(j); // 可以换成别的处理质因数
			while(n%j==0) n/=j;
		}
	}
	if(n>1){
		printf(n) // 最后剩下的n如果大于1也是一个质因数
	}
}
```

## 例题
### [B3715 分解质因子 2 - 洛谷](https://www.luogu.com.cn/problem/B3715)
```cpp
ll n;
void solve(){
	cin>>n;
	auto f=[](ll n)->void{
		for(ll j=2;j*j*1LL<=n;j++){
			if(n%j==0){
				while(n%j==0){
					cout<<j<<" ";
					n/=j;
				}
			}
		}
		if(n>1) cout<<n<<" ";
		cout<<"\n";
	};
	if(n>1){
		f(n);
	}
	else{
		cout<<1<<"\n";
	}
}
```
