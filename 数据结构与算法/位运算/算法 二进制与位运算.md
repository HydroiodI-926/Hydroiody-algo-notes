# 二进制 计数与运算
- 无符号整数： int 4字节 32bit 表示整数范围 \[0 , 2^32 -1]
- 有符号整数：
	1. int 4字节 32bit 表示整数范围 \[-2^31 , 0 , 2^31 -1\]
	2. 第一位为符号位0为正1为负 负数占一半，非负数占一半
	3. 设一正整数x ；负数表示法-x=~(x-1)  该公式具有可逆性
	4. 设一负整数x ；正数 -x=~x+1           反之亦然
	5. 特别的，对于一个有限位能表示的最小负数根据4的公式最小附属会得到本身，因此一个有限位能表示的最小负数本身值（第一位不再表示正负号）等于它的相反数，或者说有符号整数表示的最小值是对应无符号整数的最大值
	- 如此设置的优越性与不足
		- 整合基本运算运算为加运算（无情况转换）提高算力性能
		- 不能解决溢出问题
- 异或运算:
	1. 异或运算就是无进位相加
	2. 满足结合律、交换律（异或运算的结果与运算顺序无关）
	3. 0\^n=n            
	4. n\^n=0

# 例题
## 实现N皇后问题
- 用整数变量储存一维的可选状态
- 用位运算操作
```cpp
#include <bits/stdc++.h>
typedef long long ll;
typedef unsigned long long ull;
using namespace std;
const ll MAXN = 1e6;

int n,cnt=0,limit;
vector<int> res;

int gp(int n){
	int res=0;
	while(n>1){
		n>>=1;
		res++;
	}
	return res;
}

void dfs(int idx,int col,int left,int right){
	if(col==limit){
		cnt++;
		if(cnt<=3){
			for(int v:res){
				cout<<v+1<<" ";
			}
			cout<<'\n';
		}
	}
	else{
		int ban=(col|left|right);
		int cand=(limit&(~ban));
		int p=0;
		while(cand!=0){
			p=(cand&(-cand));
			res[idx]=gp(p);
			cand^=p;
			dfs(idx+1,col|p,(left|p)>>1,(right|p)<<1);
		}
	}
}

void solve() {
	cin>>n;
	res.assign(n,0);
	limit=(1<<n)-1;
	dfs(0,0,0,0);
	cout<<cnt;
}

int main(){
	ios::sync_with_stdio(0);
	cin.tie(0),cout.tie(0);
	solve();
	// int T;cin>>T;
    // while(T--){
    	// solve();
    // }
	return 0;
}
```
