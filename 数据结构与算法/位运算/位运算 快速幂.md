## [P1226 【模板】快速幂 - 洛谷](https://www.luogu.com.cn/problem/P1226)

求$a^b mod p$ 这类问题可以把它归为递归问题类似于如下：
```cpp
ll qpow(ll a,ll b,ll p){
	if(b==1) return a%p;
	if(b&1) return a*qpow(a,b>>1,p)%p*qpow(a,b>>1,p)%p;
	return qpow(a,b>>1,p)%p*qpow(a,b>>1,p)%p;
}
```
但是这么写存在大量重复计算，由此可以做优化
```cpp
ll qpow(ll a,ll b,ll p){
	if(b==0) return 1%p; // 规定任何数0次方为 1
	if(b==1) return a%p;
	ll ans=qpow(a,b>>1,p);
	ans=ans*ans%p;
	if(b&1) ans=ans*a%p
	return ans;
}
```
进一步优化，用循环迭代取代大部分递归
```cpp
ll qpow(ll a,ll b,ll p){
	ll ans=1; // 如果b==0返回1符合数学定义
	while(b){
		if(b&1) return a*qpow(a,b>>1,p)%p;
		a=a*a%p;
		b>>=1;
	}
	return ans;
}
```
最后，尝试转化为递推
```cpp
// 最终版本
ll qpow(ll a,ll b,ll p){
	ll ans=1;
	while(b){
		if(b&1) ans=ans*a%p;
		a=a*a%p;
		b>>=1;
	}
	return ans;
}
```
