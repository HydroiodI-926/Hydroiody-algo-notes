# 模板
二分答案的前提是答案范围在一个单调区间上，一般要求找符合要求的最大或者最小值（比如最小最大值），因此边界处理常常在于明确所求答案在**左闭右闭区间**还是**左闭右开区间**，一般来讲所求答案尽可能大的话会要求左闭右闭区间，打个比方如果有个最小的整数区间\[1,1]，此时左边界允许等于右边界
```cpp
	int l=0,r=MAXN,mid,ans;  // 最后答案用ans表示
	while(l<=r){
		mid=l+(r-l)/2;
		if(check(mid)){ // 检验区间应该更大还是应该更小
			ans=mid;
			l=mid+1;
		}
		else{
			r=mid-1;
		}
	}
```
另外一个是左闭右开区间，对于一个\[1,1)区间，发现此时左边界不能等于右边界：
```cpp
	int l=0,r=MAXN,mid,ans;  // 最后答案用ans表示
	while(l<r){
		mid=l+(r-l)/2;
		// 这个选的例子check返回真管右区间往回收，一般看check函数实现，一般没影响
		if(check(mid)){ 
			ans=mid;    // 不论如何，答案只在check为true里面更新
            r=mid;      // 因为l==r已经不为合法情况了，所以r的更新不需要额外减一
		}
		else{
			l=mid+1;
		}
	}
```
虽然说区间有区别，但是好像在二分答案中左闭右开区间的改成左闭右闭好像也能过，不过区间的选择与与其更新方式是一一对应的，不可混淆，不然会出现死循环

## CF2182B New Year Cake

### 题目描述
```
Monocarp is going to bake a New Year cake.

The cake must consist of at least one layer. The size of the top layer of the cake must be 1 ; the size of the layer below it must be 2; the layer below that must be 4, and so on (each layer, except for the top one, is twice the size of the layer above it).

Additionally, each layer must be covered with either white or dark chocolate. To cover a layer of size  k , Monocarp will need  k kilograms of chocolate. Each layer must be covered with exactly one type of chocolate, and these types must alternate (if some layer is covered with dark chocolate, both the layer directly below it and the layer directly above it must be covered with white chocolate, and vice versa).

Monocarp has  a  kilograms of white chocolate and b  kilograms of dark chocolate. He wants to calculate the maximum number of layers that the cake can consist of, ensuring that he has enough chocolate of both types.
```
#### 输入格式

The first line contains one integer  t  $$( 1 \le t \le 10^4 )$$ — the number of test cases.

Each test case consists of one line containing two integers  a  and  b $$ (  1 \le a, b \le 10^6  )$$.
#### 输出格式
For each test case, output one integer — the maximum possible number of layers in the cake.
#### 输入输出样例 #1

##### 输入 #1

```
7
1 1
1 2
3 1
4 3
5 2
1000000 1000000
1000000 1
```
##### 输出 #1

```
1
2
2
2
3
20
2
```
##### 说明/提示
```
In the first example, Monocarp can bake a cake with one layer of size  $ 1 $  and cover it with any type of chocolate.

In the second example, Monocarp can bake a cake with two layers: the top layer of size  $ 1 $  with white chocolate, and below it a layer of size  $ 2 $  with dark chocolate.

In the third example, Monocarp can bake a cake with two layers: the top layer of size  $ 1 $  with dark chocolate, and below it a layer of size  $ 2 $  with white chocolate.

In the fourth example, Monocarp can bake a cake with two layers: the top layer of size  $ 1 $  with dark chocolate, and below it a layer of size  $ 2 $  with white chocolate. Note that a cake with three layers, where the top layer of size  $ 1 $  and the layer below it of size  $ 2 $  are both dark chocolate, and the bottom layer of size  $ 4 $  is white chocolate, is not valid, as the types of layers must alternate.
```
### 题解
这一题是二分+等比数列+一点点贪心
```cpp
/*这道题的精髓在于check函数，算是一种比较新的写法*/
bool check(int mid,int n,int m){
	ll q=1,h=mid/2;
	for(int i=0;i<h;i++){
		q*=4;
	}
	ll sum=(q-1)/3;
	if(mid%2==0){
		ll t=sum;
		ll b=sum*2;
		return t<=n&&b<=m||t<=m&&b<=n;
	}
	else{
		ll t=(q*4-1)/3;
		ll b=sum*2;
		return t<=n&&b<=m||t<=m&&b<=n;
	}
}

void solve(){
	int n,m;cin>>n>>m;
	/* 由于涉及到等比数列之和求公比，因此上界不能太高，好在样例中展示了
	样例输入的最大值10^6的结果是20层，所以定在21就够了*/
	int l=0,r=21,mid,ans=0; 
	while(l<=r){
		mid=l+(r-l)/2;
		if(check(mid,n,m)){
			ans=mid;
			l=mid+1;
		}
		else{
			r=mid-1;
		}
	}
	cout<<ans<<"\n";
}
```
## U597587 单挑
### 题目描述
这次ACM大胃王队伍要参加大胃王比赛。比赛由 $N$ 人组成的团队为基本单位参赛，我们的队员从 $1\sim N$ 编号。第 $i$ 名队员的消化代价为 $A_i$。
比赛有 $N$ 种不同的食物，而我们的战略是让每位队员负责吃掉其中一种食物。不能有两名队员吃同一种食物，也不能让一名队员吃多与一种食物。第 $j$ 种食物的难吃程度为 $F_j$。 消化代价 $x$ 的队员吃完难吃程度 $y$ 的食物需要花费 $x\times y$ 秒。 整个队伍的成绩是 $N$ 名队员吃完食物花费时间的最大值。
比赛前，我们的队伍会进行修行。一次修行可以将一名消化代价大于 $0$ 的队员的消化代价减少 $1$。由于修行需要消耗庞大的食费，因此最多只能进行 $K$ 次修行。
通过修行和适当选择每位队员吃的食物，在比赛中能够获得的最好成绩是多少？
### 输入格式
第 $1$ 行，两个正整数 $N,K$。
第 $2$ 行，$N$ 个正整数 $A_1,A_2,\cdots,A_N$。
第 $3$ 行，$N$ 个正整数 $F_1,F_2,\cdots,F_N$。
### 输出格式

输出队伍的最好成绩(越小表示成绩越好)。
### 输入输出样例 #1

#### 输入 #1

```
3 5
4 2 1
2 3 1
```

#### 输出 #1

```
2
```

### 输入输出样例 #2

#### 输入 #2

```
3 8
4 2 1
2 3 1
```

#### 输出 #2

```
0
```

### 输入输出样例 #3

#### 输入 #3

```
11 14
3 1 4 1 5 9 2 6 5 3 5
8 9 7 9 3 2 3 8 4 6 2
```

#### 输出 #3

```
12
```


### 说明/提示
$1 \le N \le 2\times 10^5$
$0 \le K \le 10^{18}$
$1 \le A_i \le 10^6$
$1 \le F_i \le 10^6$

对于 $10\%$ 的数据，$K ≥ A_1 + A_2+...+A_n$
对于另外 $10\%$ 的数据，$N \leq 2$
对于另外 $10\%$ 的数据，$F_i = 1$
对于剩下 $70\%$ 的数据，没有限制

### 样例一解释：
$1$ 号队员进行 $4$ 次修行，吃 $2$ 号食物，花费 $0$ 秒。

$2$ 号队员进行 $1$ 次修行，吃 $3$ 号食物，花费 $1$ 秒。

$3$ 号队员进行 $0$ 次修行，吃 $1$ 号食物，花费 $2$ 秒。

总成绩取最大值 $2$ 秒。
## [管道 - 蓝桥云课(二分＋区间和并)](https://www.lanqiao.cn/problems/3544/learning/?page=1&first_category_id=1&name=%E7%AE%A1%E9%81%93)
```cpp
#include <bits/stdc++.h>
typedef long long ll;
typedef unsigned long long ull;
using namespace std;

const ll MAXN = 2e9;

ll n,m;
struct node{
	int l,s;
};

struct rag{
	ll l,r;
};

bool comp(node x,node y){
	return x.l<y.l;
}

bool comp2(rag x,rag y){
	return x.l<y.l;
}

bool check(ll mid,vector<node> &arr){
	vector<rag> b;
	for(int i=0;i<n;i++){
		if(mid>=arr[i].s){
			ll temp=mid-arr[i].s;
			ll left=max(1LL,arr[i].l-temp);
			ll right=min(m,arr[i].l+temp);
			b.push_back({left,right});
		}
	}
	if(b.empty()) return false;
	sort(b.begin(),b.end(),comp2);
	ll end=b[0].r,len=b.size();
	for(int i=1;i<len;i++){
		if(b[i].l<=end+1){
			end=max(end,b[i].r);
		}
		else{
			break;
		}
	}
	return end>=m;
}

void solve() {
	cin>>n>>m;
	vector<node> arr(n);
	for(int i=0;i<n;i++){
		cin>>arr[i].l>>arr[i].s;
	}
	sort(arr.begin(),arr.end(),comp);
	ll l=0,r=MAXN,mid,ans;
	while(l<=r){
		mid=(l+r)/2;
		if(check(mid,arr)){
			ans=mid;
			r=mid-1;
		}
		else{
			l=mid+1;
		}
	}
	cout<<ans;
}

int main(){
	ios::sync_with_stdio(0);
	cin.tie(0),cout.tie(0);
	solve();
	// int T;cin>>T;
    // while(T--){
    	// solve();
    // }
	return 0;
}
```
## [1540:舞者僵尸](https://cpc.csgrandeur.cn/csgoj/problemset/problem?pid=1540)
```cpp
#include <bits/stdc++.h>
typedef long long ll;
typedef unsigned long long ull;
using namespace std;
const ll MAXN = 2e18;
ll n,k;
struct p{
	ll t,l;
};
vector<p> a;
void solve(){
	cin>>n>>k;
	a.resize(n);
	for(int i=0;i<n;i++){
		cin>>a[i].t>>a[i].l;
	}
	sort(a.begin(),a.end(),[](p x,p y){
		return x.t<y.t;
	});
	ll left=0,right=MAXN,ans=1e12+1,mid;
	auto check=[](ll mid)->bool{
		vector<pair<ll,ll>> b;
		for(int i=0;i<n;i++){
			if(a[i].t<=mid){
				ll l=max(0LL,a[i].l-(mid-a[i].t));
				ll r=min(k,a[i].l+(mid-a[i].t));
				b.push_back({l,r});
			}
		}
		sort(b.begin(),b.end());
		if(b.empty()){
			return false;
		}
		else{
			ll last=-1;
			int len=b.size();
			for(int i=0;i<len;i++){
				if(b[i].first>last+1){
					return false;
				}
				last=max(last,b[i].second);
				if(last>=k){
					return true;
				}
			}
			return last>=k;
		}
	};
	while(left<=right){
		mid=(right-left)/2+left;
		// cout<<left<<" "<<right<<" "<<mid<<"\n";
		if(check(mid)){
			ans=mid;
			right=mid-1;
		}
		else{
			left=mid+1;
		}
	}
	cout<<ans;
}
int main(){
	ios::sync_with_stdio(0);
	cin.tie(0),cout.tie(0);
	int T=1;
	// cin>>T;
    while(T--){
    	solve();
    }
	return 0;
}
```
