## 基础整数类型

|占位符|类型|说明|示例|
|---|---|---|---|
|`%d`|`int`|有符号十进制整数|`printf("%d", 123)` → `123`|
|`%i`|`int`|有符号十进制整数(同%d)|`printf("%i", 123)` → `123`|
|`%u`|`unsigned int`|无符号十进制整数|`printf("%u", 123u)` → `123`|
|`%o`|`unsigned int`|无符号八进制整数|`printf("%o", 10)` → `12`|
|`%x`|`unsigned int`|无符号十六进制(小写)|`printf("%x", 255)` → `ff`|
|`%X`|`unsigned int`|无符号十六进制(大写)|`printf("%X", 255)` → `FF`|

## 长整数类型

|占位符|类型|说明|示例|
|---|---|---|---|
|`%ld`|`long`|长整型十进制|`printf("%ld", 123L)` → `123`|
|`%lu`|`unsigned long`|无符号长整型|`printf("%lu", 123UL)` → `123`|
|`%lld`|`long long`|长长整型|`printf("%lld", 123LL)` → `123`|
|`%llu`|`unsigned long long`|无符号长长整型|`printf("%llu", 123ULL)` → `123`|

## 浮点类型

|占位符|类型|说明|示例|
|---|---|---|---|
|`%f`|`float/double`|十进制浮点数|`printf("%f", 3.14)` → `3.140000`|
|`%lf`|`double`|双精度浮点数|`printf("%lf", 3.14)` → `3.140000`|
|`%e`|`float/double`|科学计数法(小写e)|`printf("%e", 123.45)` → `1.234500e+02`|
|`%E`|`float/double`|科学计数法(大写E)|`printf("%E", 123.45)` → `1.234500E+02`|
|`%g`|`float/double`|自动选择%f或%e|`printf("%g", 123.45)` → `123.45`|
|`%G`|`float/double`|自动选择%f或%E|`printf("%G", 123.45)` → `123.45`|
|`%a`|`float/double`|十六进制浮点数|`printf("%a", 3.14)` → `0x1.91eb8p+1`|
|`%A`|`float/double`|十六进制浮点数(大写)|`printf("%A", 3.14)` → `0X1.91EB8P+1`|

## 字符和字符串

|占位符|类型|说明|示例|
|---|---|---|---|
|`%c`|`char`|单个字符|`printf("%c", 'A')` → `A`|
|`%s`|`char*`|字符串|`printf("%s", "hello")` → `hello`|
|`%p`|`void*`|指针地址|`printf("%p", &x)` → `0x7ffe...`|

## 特殊格式

|占位符|类型|说明|示例|
|---|---|---|---|
|`%%`|-|输出百分号本身|`printf("%%")` → `%`|
|`%n`|`int*`|已输出字符数|`printf("abc%n", &count)` → `count=3`|

## 格式化修饰符

### 宽度和精度

|修饰符|说明|示例|
|---|---|---|
|`%Nd`|最小宽度N|`printf("%5d", 12)` → `12`|
|`%0Nd`|用0填充|`printf("%05d", 12)` → `00012`|
|`%.Nf`|小数点后N位|`printf("%.2f", 3.14159)` → `3.14`|
|`%*.*f`|动态宽度/精度|`printf("%*.*f", 8, 2, 3.14)` → `3.14`|

### 对齐和标志

| 修饰符    | 说明     | 示例                            |
| ------ | ------ | ----------------------------- |
| `%-Nd` | 左对齐    | `printf("%-5d", 12)` → `12`   |
| `%+d`  | 显示正负号  | `printf("%+d", 12)` → `+12`   |
| `% d`  | 正数前加空格 | `printf("% d", 12)` → `12`    |
| `%#x`  | 显示进制前缀 | `printf("%#x", 255)` → `0xff` |
