# C++ 重载详解

## 🎯 重载是什么？

**重载** 是 C++ 中的一种多态特性，允许**同一个名字**有**多个不同的实现**，编译器会根据**参数类型和数量**自动选择正确的版本。

## 💡 基本概念

### 简单比喻
想象一个"打开"动作：
- 打开(门) → 推门
- 打开(书) → 翻页  
- 打开(电脑) → 按电源键

**同一个名字"打开"，根据对象不同执行不同操作**，这就是重载的思想！

## 🔧 函数重载

### 基本示例
```cpp
#include <iostream>
using namespace std;

// 三个同名的 print 函数，但参数不同
void print(int num) {
    cout << "整数: " << num << endl;
}

void print(double num) {
    cout << "浮点数: " << num << endl;
}

void print(const string& text) {
    cout << "字符串: " << text << endl;
}

int main() {
    print(10);        // 调用 print(int)
    print(3.14);      // 调用 print(double)  
    print("Hello");   // 调用 print(const string&)
    
    return 0;
}
```
**输出：**
```
整数: 10
浮点数: 3.14
字符串: Hello
```

### 参数数量不同的重载
```cpp
#include <iostream>
using namespace std;

// 计算两个数的和
int add(int a, int b) {
    return a + b;
}

// 计算三个数的和
int add(int a, int b, int c) {
    return a + b + c;
}

// 计算四个数的和
int add(int a, int b, int c, int d) {
    return a + b + c + d;
}

int main() {
    cout << add(1, 2) << endl;       // 输出: 3
    cout << add(1, 2, 3) << endl;    // 输出: 6
    cout << add(1, 2, 3, 4) << endl; // 输出: 10
    
    return 0;
}
```

## ⚙️ 运算符重载

### 基本概念
让运算符对自定义类型也能工作：

```cpp
#include <iostream>
using namespace std;

class Point {
public:
    int x, y;
    
    Point(int x = 0, int y = 0) : x(x), y(y) {}
    
    // 重载 + 运算符
    Point operator+(const Point& other) const {
        return Point(x + other.x, y + other.y);
    }
    
    // 重载 == 运算符
    bool operator==(const Point& other) const {
        return x == other.x && y == other.y;
    }
};

int main() {
    Point p1(1, 2), p2(3, 4);
    
    Point p3 = p1 + p2;  // 使用重载的 + 运算符
    cout << "p3: (" << p3.x << ", " << p3.y << ")" << endl;
    
    if (p1 == p2) {      // 使用重载的 == 运算符
        cout << "p1 和 p2 相等" << endl;
    } else {
        cout << "p1 和 p2 不相等" << endl;
    }
    
    return 0;
}
```

## 🔍 重点：函数调用运算符重载

### 这就是我们代码中的重载！
```cpp
struct LengthCompare {
    // 重载函数调用运算符：operator()
    bool operator()(const string& a, const string& b) const {
        if (a.length() == b.length()) {
            return a < b;
        }
        return a.length() < b.length();
    }
};
```

### 理解 operator()
```cpp
// 创建比较器对象
LengthCompare comp;

// 使用函数调用运算符
bool result = comp("apple", "banana");
// 等价于：
bool result = comp.operator()("apple", "banana");
```

### 完整示例
```cpp
#include <iostream>
#include <string>
using namespace std;

struct Adder {
    // 重载函数调用运算符
    int operator()(int a, int b) const {
        return a + b;
    }
    
    string operator()(const string& a, const string& b) const {
        return a + " " + b;
    }
};

int main() {
    Adder add;
    
    // 像函数一样使用对象
    int sum = add(10, 20);
    string text = add("Hello", "World");
    
    cout << "数字和: " << sum << endl;      // 输出: 30
    cout << "字符串和: " << text << endl;   // 输出: Hello World
    
    return 0;
}
```

## 🎯 回到原代码的重载分析

```cpp
struct LengthCompare {
    bool operator()(const string& a, const string& b) const {
        if (a.length() == b.length()) {
            return a < b; // 长度相同时按字典序
        }
        return a.length() < b.length(); // 按长度升序
    }
};
```

### 这个重载做了什么？
1. **重载了 `operator()`** - 函数调用运算符
2. **使得 `LengthCompare` 对象可以像函数一样被调用**
3. **set 容器在内部排序时会"调用"这个比较器**

```cpp
// set 内部的工作方式类似：
LengthCompare comp;
if (comp(element1, element2)) {
    // element1 应该排在 element2 前面
}
```

## 📋 重载规则总结

### 可以重载的运算符
```
+  -  *  /  %  ^  &  |  ~  
!  =  <  >  +=  -=  *=  /=  
%=  ^=  &=  |=  <<  >>  >>=  
<<=  ==  !=  <=  >=  &&  ||  
++  --  ,  ->*  ->  ()  []  
new  delete  new[]  delete[]
```

### 不能重载的运算符
```
.   ::   ?:   sizeof   typeid  
.*   .->*
```

### 重载的限制
1. **不能创建新运算符**
2. **不能改变运算符的优先级**
3. **不能改变运算符的结合性**
4. **至少有一个操作数是用户自定义类型**

## 🛠️ 更多重载示例

### 输出流运算符重载
```cpp
#include <iostream>
using namespace std;

class Student {
public:
    string name;
    int age;
    
    Student(string n, int a) : name(n), age(a) {}
    
    // 重载 << 运算符，方便输出
    friend ostream& operator<<(ostream& os, const Student& s) {
        os << "学生: " << s.name << ", 年龄: " << s.age;
        return os;
    }
};

int main() {
    Student s("Alice", 20);
    cout << s << endl;  // 输出: 学生: Alice, 年龄: 20
    
    return 0;
}
```

### 数组下标运算符重载
```cpp
#include <iostream>
using namespace std;

class SafeArray {
private:
    int data[10];
public:
    // 重载 [] 运算符，添加边界检查
    int& operator[](int index) {
        if (index < 0 || index >= 10) {
            throw out_of_range("索引越界");
        }
        return data[index];
    }
};

int main() {
    SafeArray arr;
    arr[0] = 100;  // 使用重载的 []
    cout << arr[0] << endl;
    
    return 0;
}
```

## 💡 重载的优势

1. **代码更直观**：`a + b` 比 `add(a, b)` 更自然
2. **一致性**：自定义类型和内置类型用法一致
3. **灵活性**：可以为同一操作提供多种实现
4. **可读性**：让代码更接近问题领域的表达方式

## ⚠️ 使用建议

1. **保持直觉**：重载的运算符行为应该符合直觉
2. **不要滥用**：只在有意义的地方使用重载
3. **保持一致性**：相关运算符应该一起重载（如 `==` 和 `!=`）
4. **遵循惯例**：`+` 不应该修改操作数，而 `+=` 应该

## 🎯 核心要点

**重载 = 同一个名字 + 不同实现 + 自动选择**

在之前的代码中：
- **重载了 `operator()`** 
- **让结构体对象可以像函数一样调用**
- **set 容器利用这个特性进行自定义排序**
