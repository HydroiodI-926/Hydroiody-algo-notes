#include <iostream>
#include <set>
#include <string>

// 自定义比较函数：按长度排序，长度相同则按字典序
struct LengthCompare {
    bool operator()(const string& a, const string& b) const {
        if (a.length() == b.length()) {

        return a.length() < b.length(); // 按长度升序
    }

## 🎯 排序原理详解

### set 的排序机制
```cpp
// set 的内部工作方式类似于：
if (comparator(a, b)) {
    // a 应该排在 b 前面
} else if (comparator(b, a)) {
    // b 应该排在 a 前面
} else {
    // a 和 b 相等（根据比较规则）
}
```

### 我们的比较规则执行过程
对于输入：`{"apple", "banana", "cat", "dog", "elephant", "fox"}`

1. **比较 "cat" 和 "dog"**
   - 长度都是 3，相等
   - 按字典序：`"cat" < "dog"` → `true`
   - 结果："cat" 排在 "dog" 前面

2. **比较 "cat" 和 "apple"**
   - 长度：3 vs 5 → `3 < 5` → `true`
   - 结果："cat" 排在 "apple" 前面

3. **最终排序逻辑**
   - 先按长度从小到大
   - 长度相同时按字典序
