# C++ 比较器详解

## 🎯 **比较器是什么？**

**比较器**是一个告诉程序"如何比较两个元素"的函数或对象，用于定义排序规则、集合顺序等。

## 🔧 **比较器的核心规则**

### 基本定义
```cpp
// 比较函数返回 bool 值：
// true  - a 应该在 b 前面
// false - a 不应该在 b 前面
bool compare(const T& a, const T& b) {
    return ...; // 你的比较逻辑
}
```

## 💡 **三种比较器形式**

### 1. 普通函数
```cpp
bool compareInt(int a, int b) {
    return a < b;  // 升序
}

// 使用
sort(arr.begin(), arr.end(), compareInt);
```

### 2. 函数对象（仿函数）
```cpp
struct CompareInt {
    bool operator()(int a, int b) const {
        return a < b;
    }
};

// 使用
sort(arr.begin(), arr.end(), CompareInt());
set<int, CompareInt> mySet;
```

### 3. Lambda 表达式
```cpp
// 使用
sort(arr.begin(), arr.end(), [](int a, int b) {
    return a < b;
});
```

## 🛠️ **详细示例分析**

### 示例1：理解返回值含义
```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main() {
    vector<int> nums = {5, 2, 8, 1, 9};
    
    // 升序比较器
    sort(nums.begin(), nums.end(), [](int a, int b) {
        cout << "比较 " << a << " 和 " << b << " -> ";
        bool result = a < b;
        cout << result << endl;
        return result;
    });
    
    cout << "排序结果: ";
    for (int n : nums) cout << n << " ";
    cout << endl;
    
    return 0;
}
```

**关键理解：**
- `return a < b` → 升序（小的在前）
- `return a > b` → 降序（大的在前）

### 示例2：自定义对象排序
```cpp
#include <iostream>
#include <algorithm>
#include <vector>
#include <string>
using namespace std;

struct Student {
    string name;
    int score;
    int age;
};

int main() {
    vector<Student> students = {
        {"Alice", 85, 20},
        {"Bob", 92, 19},
        {"Charlie", 85, 21},
        {"Diana", 78, 20}
    };
    
    // 单条件排序：按分数降序
    sort(students.begin(), students.end(), [](const Student& a, const Student& b) {
        return a.score > b.score;  // 分数高的在前
    });
    
    cout << "按分数降序:" << endl;
    for (const auto& s : students) {
        cout << s.name << " " << s.score << endl;
    }
    
    return 0;
}
```

### 示例3：多条件排序
```cpp
// 多条件排序：先按分数降序，分数相同按年龄升序
sort(students.begin(), students.end(), [](const Student& a, const Student& b) {
    if (a.score == b.score) {
        return a.age < b.age;  // 分数相同时，年龄小的在前
    }
    return a.score > b.score;  // 分数高的在前
});

cout << "多条件排序:" << endl;
for (const auto& s : students) {
    cout << s.name << " 分数:" << s.score << " 年龄:" << s.age << endl;
}
```

## 📋 **在 STL 容器中的应用**

### set/map 中的比较器
```cpp
#include <iostream>
#include <set>
#include <string>
using namespace std;

// 按字符串长度排序
struct LengthCompare {
    bool operator()(const string& a, const string& b) const {
        return a.length() < b.length();
    }
};

int main() {
    set<string, LengthCompare> words = {
        "apple", "banana", "cat", "elephant"
    };
    
    for (const auto& word : words) {
        cout << word << " ";  // 输出: cat apple banana elephant
    }
    
    return 0;
}
```

### priority_queue 中的比较器
```cpp
#include <iostream>
#include <queue>
#include <vector>
using namespace std;

// 小顶堆比较器
struct MinHeapCompare {
    bool operator()(int a, int b) const {
        return a > b;  // 注意：priority_queue 是反直觉的！
    }
};

int main() {
    // 小顶堆
    priority_queue<int, vector<int>, MinHeapCompare> minHeap;
    
    minHeap.push(5);
    minHeap.push(2);
    minHeap.push(8);
    
    cout << "小顶堆顶部: " << minHeap.top() << endl;  // 输出: 2
    
    return 0;
}
```

## 🎯 **比较器设计模式**

### 模式1：属性比较
```cpp
// 按对象的某个属性排序
sort(students.begin(), students.end(), 
    [](const Student& a, const Student& b) {
        return a.score < b.score;  // 按分数属性
    });
```

### 模式2：复合条件
```cpp
// 多个条件组合
sort(students.begin(), students.end(), 
    [](const Student& a, const Student& b) {
        // 先按分数降序，再按姓名升序
        if (a.score != b.score) {
            return a.score > b.score;
        }
        return a.name < b.name;
    });
```

### 模式3：自定义规则
```cpp
// 特殊排序规则：偶数在前，奇数在后，各自升序
sort(numbers.begin(), numbers.end(), 
    [](int a, int b) {
        bool a_even = (a % 2 == 0);
        bool b_even = (b % 2 == 0);
        
        if (a_even != b_even) {
            return a_even;  // 偶数在前
        }
        return a < b;       // 同奇偶性时升序
    });
```

## ⚠️ **重要注意事项**

### 1. 严格弱序规则
比较器必须满足：
- **非自反性**：`compare(a, a)` 必须为 false
- **非对称性**：如果 `compare(a, b)` 为 true，则 `compare(b, a)` 必须为 false
- **传递性**：如果 `compare(a, b)` 和 `compare(b, c)` 为 true，则 `compare(a, c)` 必须为 true

### 2. 错误示例
```cpp
// ❌ 错误：违反严格弱序
bool badCompare(int a, int b) {
    return a <= b;  // 违反了非自反性
}

// ✅ 正确
bool goodCompare(int a, int b) {
    return a < b;   // 满足严格弱序
}
```

### 3. 相等处理
```cpp
// 在 set/map 中，如果 !compare(a,b) && !compare(b,a)，认为 a 和 b 相等
// 相等的元素不会被重复插入

set<int, CompareInt> s;
s.insert(5);
s.insert(5);  // 不会被插入，因为 5 和 5 "相等"
```

## 🔧 **实用技巧**

### 技巧1：可配置的比较器
```cpp
struct ConfigurableCompare {
    bool ascending;
    
    ConfigurableCompare(bool asc = true) : ascending(asc) {}
    
    bool operator()(int a, int b) const {
        return ascending ? (a < b) : (a > b);
    }
};

// 使用
sort(arr.begin(), arr.end(), ConfigurableCompare(true));   // 升序
sort(arr2.begin(), arr2.end(), ConfigurableCompare(false)); // 降序
```

### 技巧2：复用标准比较器
```cpp
#include <functional>

// 使用标准库的比较器
sort(arr.begin(), arr.end(), greater<int>());    // 降序
sort(arr.begin(), arr.end(), less<int>());       // 升序（默认）
```

### 技巧3：成员函数指针比较器
```cpp
#include <algorithm>
#include <vector>
#include <string>

struct Person {
    string name;
    int getAge() const { return age; }
private:
    int age = 20;
};

// 使用成员函数作为比较器
vector<Person> people;
sort(people.begin(), people.end(),
    [](const Person& a, const Person& b) {
        return a.getAge() < b.getAge();
    });
```

## 💡 **性能考虑**

### 内联优化
```cpp
// Lambda 表达式通常能被编译器内联，性能较好
sort(arr.begin(), arr.end(), [](int a, int b) {
    return a < b;  // 可能被内联
});

// 函数对象也容易被内联
struct Compare {
    bool operator()(int a, int b) const { return a < b; }
};
```

### 避免昂贵操作
```cpp
// ❌ 不好：在比较器中执行昂贵操作
sort(strings.begin(), strings.end(), 
    [](const string& a, const string& b) {
        return a.length() + computeSomething(a) < 
               b.length() + computeSomething(b);
    });

// ✅ 好：预计算昂贵值
vector<pair<int, string>> computed;
for (const auto& s : strings) {
    computed.emplace_back(s.length() + computeSomething(s), s);
}
sort(computed.begin(), computed.end());
```

## 📋 **总结**

### 比较器的核心要点：
1. **返回值意义**：`true` 表示 a 应该在 b 前面
2. **三种形式**：函数、函数对象、Lambda
3. **严格弱序**：必须满足数学规则
4. **广泛应用**：sort、set、map、priority_queue 等

### 记住这个模式：
```cpp
bool compare(const T& a, const T& b) {
    // 定义你的排序规则
    // return true  if a should come before b
    // return false if a should not come before b
}
```

**一句话总结：比较器就是定义元素顺序规则的"裁判"！**
