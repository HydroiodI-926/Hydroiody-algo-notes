# `->` 运算符详解

## 基本概念
`->` 是 C++ 中的**成员访问运算符**，用于通过**指针**访问对象的成员（变量或函数）。

## 简单理解
`->` 就像是**指针的"点运算符"**：
- 对象用 `.` 访问成员：`对象.成员`
- 指针用 `->` 访问成员：`指针->成员`

## 基本语法
```cpp
指针变量->成员名
```

## 与 `.` 运算符的对比

### 使用对象（用 `.`）
```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student {
    string name;
    int score;
    
    void printInfo() {
        cout << name << ": " << score << endl;
    }
};

int main() {
    Student s;          // s是对象
    s.name = "张三";    // 用 . 访问成员
    s.score = 95;
    s.printInfo();      // 用 . 调用成员函数
    
    return 0;
}
```

### 使用指针（用 `->`）
```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student {
    string name;
    int score;
    
    void printInfo() {
        cout << name << ": " << score << endl;
    }
};

int main() {
    Student s;
    Student* ptr = &s;  // ptr是指向Student的指针
    
    ptr->name = "李四";  // 用 -> 访问成员
    ptr->score = 88;
    ptr->printInfo();    // 用 -> 调用成员函数
    
    return 0;
}
```

## 为什么需要 `->` 运算符？

### 内存角度理解
```cpp
Student s;              // s是实际对象
Student* ptr = &s;      // ptr存储的是s的地址

// 如果用 . 访问指针成员会怎样？
// ptr.name = "王五";   // ❌ 错误！ptr是地址，不是对象

// 正确方式：先解引用，再用 .
(*ptr).name = "王五";   // ✅ 正确，但繁琐

// 更简洁的方式：用 ->
ptr->name = "王五";     // ✅ 正确，且简洁
```

**`->` 实际上是语法糖：**
```cpp
ptr->成员  等价于  (*ptr).成员
```

## 实际应用场景

### 场景1：动态对象
```cpp
#include <iostream>
#include <string>
using namespace std;

class Person {
public:
    string name;
    int age;
    
    void introduce() {
        cout << "我是" << name << "，今年" << age << "岁" << endl;
    }
};

int main() {
    // 动态创建对象（在堆上）
    Person* personPtr = new Person();
    
    // 通过指针访问成员
    personPtr->name = "小明";
    personPtr->age = 20;
    personPtr->introduce();  // 输出：我是小明，今年20岁
    
    delete personPtr;  // 记得释放内存
    return 0;
}
```

### 场景2：链表节点访问
```cpp
#include <iostream>
using namespace std;

struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

int main() {
    // 创建链表: 1 -> 2 -> 3
    ListNode* head = new ListNode(1);
    head->next = new ListNode(2);
    head->next->next = new ListNode(3);
    
    // 遍历链表
    ListNode* current = head;
    while (current != nullptr) {
        cout << current->val << " -> ";  // 用 -> 访问val
        current = current->next;         // 用 -> 访问next
    }
    cout << "nullptr" << endl;
    
    return 0;
}
```

### 场景3：STL 迭代器
```cpp
#include <iostream>
#include <vector>
#include <map>
using namespace std;

int main() {
    // vector 迭代器
    vector<int> vec = {1, 2, 3, 4, 5};
    for (auto it = vec.begin(); it != vec.end(); it++) {
        cout << *it << " ";  // 迭代器用 * 解引用
    }
    cout << endl;
    
    // map 迭代器
    map<string, int> scores = {{"Alice", 95}, {"Bob", 87}};
    for (auto it = scores.begin(); it != scores.end(); it++) {
        // map迭代器指向pair，用->访问first和second
        cout << it->first << ": " << it->second << endl;
    }
    
    return 0;
}
```

## 详细示例分析

### 示例1：结构体指针
```cpp
#include <iostream>
using namespace std;

struct Point {
    int x;
    int y;
    
    void display() {
        cout << "(" << x << ", " << y << ")" << endl;
    }
};

int main() {
    Point p = {10, 20};
    Point* ptr = &p;
    
    // 三种访问方式对比：
    
    // 1. 通过对象访问（用 .）
    cout << p.x << ", " << p.y << endl;
    p.display();
    
    // 2. 通过指针先解引用（用 * 和 .）
    cout << (*ptr).x << ", " << (*ptr).y << endl;
    (*ptr).display();
    
    // 3. 通过指针直接访问（用 ->）
    cout << ptr->x << ", " << ptr->y << endl;
    ptr->display();
    
    return 0;
}
```

### 示例2：类指针
```cpp
#include <iostream>
using namespace std;

class Calculator {
private:
    double result;
public:
    Calculator() : result(0) {}
    
    void add(double x) {
        result += x;
    }
    
    void multiply(double x) {
        result *= x;
    }
    
    double getResult() {
        return result;
    }
};

int main() {
    Calculator calc;
    Calculator* calcPtr = &calc;
    
    // 通过对象操作
    calc.add(5);
    calc.multiply(3);
    cout << "结果: " << calc.getResult() << endl;  // 15
    
    // 通过指针操作
    calcPtr->add(10);
    calcPtr->multiply(2);
    cout << "结果: " << calcPtr->getResult() << endl;  // 50
    
    return 0;
}
```

## 嵌套访问

### 多层指针访问
```cpp
#include <iostream>
using namespace std;

struct Node {
    int data;
    Node* next;
};

int main() {
    Node* head = new Node{1, new Node{2, new Node{3, nullptr}}};
    
    // 多层 -> 访问
    cout << head->data << endl;           // 1
    cout << head->next->data << endl;     // 2
    cout << head->next->next->data << endl; // 3
    
    return 0;
}
```

### 指针数组访问
```cpp
#include <iostream>
using namespace std;

struct Student {
    string name;
    int score;
};

int main() {
    const int SIZE = 3;
    Student* students[SIZE];  // 指针数组
    
    // 动态创建学生对象
    for (int i = 0; i < SIZE; i++) {
        students[i] = new Student();
        students[i]->name = "学生" + to_string(i + 1);
        students[i]->score = 80 + i * 5;
    }
    
    // 访问指针数组中的对象
    for (int i = 0; i < SIZE; i++) {
        cout << students[i]->name << ": " << students[i]->score << endl;
    }
    
    // 释放内存
    for (int i = 0; i < SIZE; i++) {
        delete students[i];
    }
    
    return 0;
}
```

## 常见错误

### 错误1：对非指针使用 `->`
```cpp
struct Point { int x, y; };

Point p;
// p->x = 10;  // ❌ 错误！p是对象，不是指针
p.x = 10;      // ✅ 正确
```

### 错误2：空指针使用 `->`
```cpp
struct Point { int x, y; };

Point* ptr = nullptr;
// ptr->x = 10;  // ❌ 错误！空指针访问，程序崩溃
```

### 错误3：混淆 `->` 和 `.`
```cpp
struct Point { int x, y; };

Point p;
Point* ptr = &p;

// ❌ 混淆使用
// ptr.x = 10;    // 错误：指针用了点
// p->x = 10;     // 错误：对象用了箭头

// ✅ 正确使用
p.x = 10;        // 对象用点
ptr->x = 10;     // 指针用箭头
```

## 在算法竞赛中的应用

### 动态数据结构
```cpp
// 树节点访问
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

void traverse(TreeNode* root) {
    if (root == nullptr) return;
    
    // 用 -> 访问树节点
    cout << root->val << " ";
    traverse(root->left);
    traverse(root->right);
}
```

### STL 容器遍历
```cpp
#include <iostream>
#include <map>
#include <vector>
using namespace std;

int main() {
    // map遍历（迭代器指向pair）
    map<string, int> m = {{"a", 1}, {"b", 2}};
    for (auto it = m.begin(); it != m.end(); it++) {
        cout << it->first << " -> " << it->second << endl;
    }
    
    // vector of pointers
    vector<Point*> points = {new Point{1,2}, new Point{3,4}};
    for (Point* p : points) {
        cout << p->x << ", " << p->y << endl;  // 用->访问
    }
    
    return 0;
}
```

## 总结

### `->` 运算符的核心要点：
1. **用于指针**访问成员
2. **等价于** `(*指针).成员`
3. **更简洁** than 先解引用再用点

### 使用规则：
- **对象**用 `.` ：`对象.成员`
- **指针**用 `->` ：`指针->成员`

### 记忆口诀：
> "点(.)用于对象，箭头(->)用于指针"  
> "指针箭头指成员，对象圆点访问它"

### 重要提醒：
- 确保指针不是 `nullptr` 再使用 `->`
- 动态分配的对象记得用 `delete` 释放
