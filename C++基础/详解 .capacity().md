## 📌 基本概念
- `capacity()` 返回vector**当前分配的内存容量**
- 表示vector**最多能容纳多少元素**而不需要重新分配内存
- 单位是**元素个数**，不是字节数

## 🎯 语法格式
```cpp
size_t current_capacity = v.capacity();
```

## 📝 使用示例
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v;
    
    cout << "初始容量: " << v.capacity() << endl;  // 0
    
    v.push_back(1);
    cout << "添加1个元素后容量: " << v.capacity() << endl;  // 1
    
    v.push_back(2);
    cout << "添加2个元素后容量: " << v.capacity() << endl;  // 2
    
    v.push_back(3);
    cout << "添加3个元素后容量: " << v.capacity() << endl;  // 4
    
    return 0;
}
```

## 🔍 容量 vs 大小
```cpp
vector<int> v = {1, 2, 3, 4, 5};

cout << "大小(size): " << v.size() << endl;      // 5 (实际元素个数)
cout << "容量(capacity): " << v.capacity() << endl; // >=5 (分配的空间)
```

**比喻理解：**
- **size** = 杯子里的水量
- **capacity** = 杯子的总容量

## 📊 扩容机制
vector的容量会**自动增长**，通常策略：
```
初始: capacity = 0
添加第1个元素: capacity = 1  
添加第2个元素: capacity = 2
添加第3个元素: capacity = 4 (翻倍)
添加第5个元素: capacity = 8 (翻倍)
添加第9个元素: capacity = 16 (翻倍)
```

## 🛠️ 实用操作

### 预分配容量（提高性能）
```cpp
vector<int> v;
v.reserve(1000);  // 预先分配1000个元素的空间
cout << v.capacity();  // 1000
```

### 查看容量使用率
```cpp
vector<int> v = {1, 2, 3, 4, 5};
double usage = (double)v.size() / v.capacity() * 100;
cout << "容量使用率: " << usage << "%" << endl;
```

### 缩减容量
```cpp
vector<int> v(1000);
v.resize(10);  // 大小变为10，但容量可能还是1000

v.shrink_to_fit();  // 请求缩减容量到刚好容纳当前元素
cout << v.capacity();  // 可能变为10（或接近10）
```

## ⚡ 性能提示
- **频繁`push_back`时**：先用`reserve()`预分配，避免多次扩容
- **内存敏感场景**：用`shrink_to_fit()`释放多余容量
