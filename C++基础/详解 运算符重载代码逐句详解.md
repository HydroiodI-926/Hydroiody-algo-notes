# 运算符重载代码逐句详解

## 🔍 代码逐行解析

### 第1-2行：头文件和命名空间
```cpp
#include <iostream>
using namespace std;
```
- `#include <iostream>`：包含输入输出流库，提供 `cout`、`cin` 等功能
- `using namespace std`：使用标准命名空间，这样就不用写 `std::cout`，可以直接写 `cout`

### 第4-6行：类定义开始
```cpp
class Point {
public:
    int x, y;
```
- `class Point`：定义一个名为 `Point` 的类（表示二维坐标点）
- `public`：访问修饰符，表示后面的成员是公开的，可以在类外部访问
- `int x, y`：声明两个整数成员变量，表示点的 x 坐标和 y 坐标

### 第8行：构造函数
```cpp
    Point(int x = 0, int y = 0) : x(x), y(y) {}
```
- `Point(int x = 0, int y = 0)`：构造函数，参数有默认值（都是0）
- `: x(x), y(y)`：初始化列表，用参数初始化成员变量
- `{}`：空的构造函数体（因为初始化列表已经完成了所有工作）

**构造函数的作用：**
```cpp
Point p1;           // 创建点 (0, 0)
Point p2(5);        // 创建点 (5, 0)  
Point p3(1, 2);     // 创建点 (1, 2)
```

### 第11-13行：重载 + 运算符
```cpp
    // 重载 + 运算符
    Point operator+(const Point& other) const {
        return Point(x + other.x, y + other.y);
    }
```
[[详解 重载+运算符代码样例]]

**逐词解析：**
- `Point`：返回类型，返回一个新的 Point 对象
- `operator+`：重载 + 运算符
- `(const Point& other)`：参数，常量引用（避免拷贝，保护原对象不被修改）
- `const`：表示这个函数不会修改当前对象
- `return Point(x + other.x, y + other.y)`：创建新点，坐标是两个点的坐标和

**使用效果：**
```cpp
Point p1(1, 2), p2(3, 4);
Point p3 = p1 + p2;  // p3 变成 (4, 6)
// 等价于：Point p3 = p1.operator+(p2);
```

### 第15-17行：重载 == 运算符
```cpp
    // 重载 == 运算符
    bool operator==(const Point& other) const {
        return x == other.x && y == other.y;
    }
```

**逐词解析：**
- `bool`：返回类型，返回比较结果（true 或 false）
- `operator==`：重载 == 运算符
- `(const Point& other)`：要比较的另一个点
- `const`：不修改当前对象
- `return x == other.x && y == other.y`：两个点的 x 和 y 都相等时才返回 true

**使用效果：**
```cpp
Point a(1, 2), b(1, 2), c(3, 4);
if (a == b) {  // true，调用 a.operator==(b)
if (a == c) {  // false
```

### 第19行：类定义结束
```cpp
};
```
- 结束 Point 类的定义

### 第21-23行：主函数开始和对象创建
```cpp
int main() {
    Point p1(1, 2), p2(3, 4);
```
- `int main()`：程序入口函数
- `Point p1(1, 2), p2(3, 4)`：创建两个 Point 对象
  - p1：坐标为 (1, 2)
  - p2：坐标为 (3, 4)

### 第25-26行：使用重载的 + 运算符
```cpp
    Point p3 = p1 + p2; // 使用重载的 + 运算符
    cout << "p3: (" << p3.x << ", " << p3.y << ")" << endl;
```
- `p1 + p2`：使用重载的 + 运算符，实际调用 `p1.operator+(p2)`
- 计算结果：p3 = (1+3, 2+4) = (4, 6)
- 输出 p3 的坐标

**注意：原代码有错误**，应该是：
```cpp
cout << "p3: (" << p3.x << ", " << p3.y << ")" << endl;
// 而不是原代码中的：cout << "p3: (' '< p3.x << ", " << p3.y << ")" << endl;
```

### 第28-33行：使用重载的 == 运算符
```cpp
    if (p1 == p2) {    // 使用重载的 == 运算符
        cout << "p1 和 p2 相等" << endl;
    } else {
        cout << "p1 和 p2 不相等" << endl;
    }
```
- `p1 == p2`：使用重载的 == 运算符，实际调用 `p1.operator==(p2)`
- 比较结果：p1(1,2) 不等于 p2(3,4)，所以执行 else 分支
- 输出："p1 和 p2 不相等"

### 第35-36行：程序结束
```cpp
    return 0;
}
```
- `return 0`：程序正常结束
- `}`：主函数结束

## 🛠️ 修正后的完整代码

```cpp
#include <iostream>
using namespace std;

class Point {
public:
    int x, y;

    Point(int x = 0, int y = 0) : x(x), y(y) {}

    // 重载 + 运算符
    Point operator+(const Point& other) const {
        return Point(x + other.x, y + other.y);
    }

    // 重载 == 运算符
    bool operator==(const Point& other) const {
        return x == other.x && y == other.y;
    }
};

int main() {
    Point p1(1, 2), p2(3, 4);

    Point p3 = p1 + p2; // 使用重载的 + 运算符
    cout << "p3: (" << p3.x << ", " << p3.y << ")" << endl;  // 修正输出语句

    if (p1 == p2) {    // 使用重载的 == 运算符
        cout << "p1 和 p2 相等" << endl;
    } else {
        cout << "p1 和 p2 不相等" << endl;
    }

    return 0;
}
```

## 📊 程序输出
```
p3: (4, 6)
p1 和 p2 不相等
```

## 💡 关键知识点总结

1. **运算符重载语法**：`返回类型 operator运算符(参数) const`
2. **成员函数重载**：左操作数是当前对象，右操作数是参数
3. **const 的作用**：保证函数不修改对象状态
4. **引用参数**：`const Point&` 避免拷贝，提高效率
5. **自然语法**：重载后可以使用 `p1 + p2` 这样的直观语法

## 🎯 重载的核心思想

**让自定义类型像内置类型一样工作！**

- 内置类型：`int a=1, b=2; int c = a + b;`
- 自定义类型：`Point p1(1,2), p2(3,4); Point p3 = p1 + p2;`

通过运算符重载，让代码更直观、更易读！
